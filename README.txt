Телеграм-бот nuhw15bot выводит курсы валют с сайта Центрального банка России на любую запрошенную дату.

Дата вводится командой /date ДД.ММ.ГГГ   Введенная дата сохраняется для последующих запросов,
и может быть изменена  следующей командой /date. Дата в неправильном формате не учитывается,
вместо нее используется текущая дата

Коды валют вводятся после конанды /curr, можно указать несколько кодов через пробел. Например /curr USD EUR

Коды некоторых валют:
USD доллар США
EUR евро
GBP фунт стерлингов Великобритании
BYN белорусский рубль
CAD канадский доллар
CHF швейцарский франк
JPY японская иена

Команда /start выводит приветсвие бота с краткой справкой по командам.
Команда /help выводит подробную справку по командам.
